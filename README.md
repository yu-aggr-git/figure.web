# figure.web

フィギュア写真の投稿・閲覧、フィギュア発売情報の共通サービスです。<br>


## 概要
* ゲストユーザー<br>
　フィギュア写真の閲覧、フィギュア発売日の閲覧のみ<br>

* 登録ユーザー<br>
  フィギュア写真の投稿、他ユーザーの写真をお気に入り追加、フィギュア発売日のお気に入り追加<br>
 
* 管理ユーザー<br>
  発売情報の登録、お知らせ機能、お問い合わせ一覧の閲覧<br>
